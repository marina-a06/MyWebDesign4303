# Mom Life Command Center

## Project Overview

This web app will help busy parents manage everything in one place. 

### Main Features

* Family Calendar
* To-Do Lists
* Meal Planner + Grocery list generator
* Kid Activity Tracker
* Budget Tracker


## What is this?
The Mom Life Command Center is a personal dashboard-style website designed to help organize and simplify daily family life. It includes a central hub for navigating important areas like schedules, reminders, tasks, and family updates in one easy-to-use interface. The goal is to make everyday planning more visual, structured, and stress-free.

## Why does it exist?
This project was created to solve the problem of managing multiple responsibilities and information sources in one place. Instead of relying on sticky notes, calendars, and scattered apps, this website brings everything together into a single organized system. It is designed to help improve time management, reduce mental clutter, and make daily planning more efficient for busy family life.

## What tools did I use?
This project was built using:
- HTML – to structure the website content
- CSS – to style and design the user interface
- Visual Studio Code – as the code editor for building the project
- GitHub – for version control and hosting the repository
- GitHub Pages – for deploying the live website
- AI tools (ChatGPT, Google, Google Gemini) – for guidance, debugging support, and improving code structure
